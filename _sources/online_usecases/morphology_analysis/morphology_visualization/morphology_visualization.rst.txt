########################
Morphology Visualization
########################

*Morphology Visualization* use case provides capability to visualize neuronal morphology in 3D inside of a Browser window.

1. The use case can be found in the *Online Use Cases/Morphology Analysis*:

     .. image:: images/morphology_analysis.png
        :width: 700px

2. When you arrive at the *Morphology Analysis* use case you can select the single cell morphology from the HBP data:

     .. image:: images/morphology_analysis_select.png
        :width: 700px

3. As a result the single cell morphology will open in a new Browser window ready for visual inspection:

     .. image:: images/morphology_analysis_view.png
        :width: 700px
